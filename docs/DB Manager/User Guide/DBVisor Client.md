# DBVisor Client

The primary objective of the DBVisor Client is to offer a straightforward method for downloading processed dumps. This versatile tool can be seamlessly integrated into various toolboxes for tasks such as automatic project setup, incorporation into Docker tools, or used as a standalone application. Its flexibility allows developers to efficiently manage and access the latest processed database dumps, enhancing convenience in project workflows.

Follow [link](https://outline.bridge.digital./Getting%20Started/Installation.md#h-client-side) to get installation instructions.

To get list of all available command type in you console the following command:

```bash
db-manager list
```

Also, you can use `help` command to get more instructions:

```bash
db-manager help
```
